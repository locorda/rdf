Converts a set of unmapped triples to a Dart object.

This method processes the given triples that were not mapped to any specific property or type and converts them into a Dart object.

triples The list of triples to convert context The current deserialization context

Returns the resulting Dart object

Implementation

@override
Map<IriTerm, List<RdfObject>> fromUnmappedTriples(Iterable<Triple> triples) {
  final result = <IriTerm, List<RdfObject>>{};
  for (final triple in triples) {
    final predicate = triple.predicate;
    final object = triple.object;
    if (predicate is! IriTerm) {
      throw ArgumentError(
          'Expected predicate to be of type IriTerm, but got $predicate');
    }
    result.putIfAbsent(predicate, () => []).add(object);
  }
  return result;
}
