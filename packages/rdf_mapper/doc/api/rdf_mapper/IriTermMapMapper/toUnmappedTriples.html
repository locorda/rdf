Converts a Dart object to a set of unmapped triples.

This method processes the given object and converts it into a list of triples that are not mapped to any specific property or type. This is useful for handling additional data that doesn't fit into the standard serialization model.

subject The subject IRI of the object being serialized value The object to convert

Returns the resulting list of triples

Implementation

@override
Iterable<Triple> toUnmappedTriples(
    RdfSubject subject, Map<RdfPredicate, List<RdfObject>> value) {
  return value.entries
      .expand((e) => e.value.map((o) => Triple(subject, e.key, o)));
}
